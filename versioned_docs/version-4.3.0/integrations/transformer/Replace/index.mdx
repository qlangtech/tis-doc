## Regex Replace

<Figure img={require('./RegexPatternReplace.png')}/>

* **配置项说明:** 

<PluginFields>

1. from

	* **类型:** 单选
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		从下拉列表中选择记录中的某一列，作为需要转换的值来源

2. 模式

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 

		使用正则式匹配目标字段内容，例如，现有数据列为 `yyyy-MM-dd` 格式，用户希望将内容格式变成 `yyyy-MM`，则可以进行如下设置：
		
		  将`模式`内容设置为`^(\d{4}-\d{2})-\d{2}$`，替换内容内容设置为`$1`

3. 替换内容

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 

		例如，现有数据列为 `yyyy-MM-dd` 格式，用户希望将内容格式变成 `yyyy-MM`，则可以进行如下设置：
		
		  将`模式`内容设置为`^(\d{4}-\d{2})-\d{2}$`，替换内容内容设置为`$1`

4. throwErrorNotMatch

	* **类型:** 单选
	* **必须:** 是
	* **默认值:** false
	* **说明:** 		如果模式没有匹配，则主动抛出异常终止处理


</PluginFields>
