## DingTalk_Chan

<Figure img={require('./DingTalkAlertChannel.png')}/>

* **配置项说明:** 

<PluginFields>

1. name

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		实例的唯一性名称，取一个方便记忆的名称

2. Access Token

	* **类型:** 密码
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		钉钉机器人的Webhook URL中的access_token参数值。创建机器人后在Webhook地址中获取

3. 加签密钥

	* **类型:** 密码
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		钉钉机器人的加签密钥(Secret)，用于验证请求合法性。如果机器人启用了加签功能则必填

4. @所有人

	* **类型:** 单选
	* **必须:** 否
	* **默认值:** false
	* **说明:** 		是否在消息中@所有群成员

5. @指定成员

	* **类型:** 单行文本
	* **必须:** 否
	* **默认值:** 无
	* **说明:** 		需要@的成员手机号，多个手机号用英文逗号分隔。被@的人会收到额外的消息提醒

6. 模版

	* **类型:** 富文本
	* **必须:** 是
	* **默认值:** com.qlangtech.tis.plugin.alert.impl.DingTalkAlertChannel.loadDefaultTpl()
	* **说明:** 		报警信息模版(velocity)，请谨慎修改避免不必要的错误。可用变量: title, subject, jobName, status, startTime, endTime, duration, link, restart, restartIndex, totalRestart等


</PluginFields>
