## Email_Chan

<Figure img={require('./EmailAlertChannel.png')}/>

* **配置项说明:** 

<PluginFields>

1. name

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		实例的唯一性名称，取一个方便记忆的名称

2. SMTP服务器

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		邮件服务器地址，例如: smtp.qq.com, smtp.163.com, smtp.gmail.com

3. SMTP端口

	* **类型:** 整型数字
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		邮件服务器端口，通常为25(不加密)、465(SSL)、587(TLS)

4. 用户名

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		SMTP认证用户名，通常为完整的邮箱地址

5. 密码

	* **类型:** 密码
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		SMTP认证密码或授权码(某些邮箱需要使用专用授权码)

6. 发件人地址

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		发件人邮箱地址，必须与SMTP认证用户名匹配

7. 收件人地址

	* **类型:** 单行文本
	* **必须:** 是
	* **默认值:** 无
	* **说明:** 		收件人邮箱地址，多个收件人用英文逗号分隔

8. 启用SSL

	* **类型:** 单选
	* **必须:** 是
	* **默认值:** false
	* **说明:** 		是否使用SSL加密连接(端口465通常需要启用)

9. 模版

	* **类型:** 富文本
	* **必须:** 是
	* **默认值:** com.qlangtech.tis.plugin.alert.impl.EmailAlertChannel.loadDefaultTpl()
	* **说明:** 		报警信息模版(velocity)，请谨慎修改避免不必要的错误。可用变量: title, subject, jobName, status, startTime, endTime, duration, link, restart, restartIndex, totalRestart等


</PluginFields>
